Getting Started

Install IoT.JS

$ npm install iotjs

Send PM2.5 Data to Data Cloud

Code for a PM2.5 air quality station to send data to a data cloud service.

var iotjs = require('iotjs');
var request = require('request');
// Define an air quality sensor instance.
var aq = new iotjs.Sensor.AirQuality({ type: 'pms3003', rate: 60000 });

// Send data to data cloud.
aq.on('data', funciton(data) {
  request.post('REST_API_URI', { form: { data: data } });
});

Get Sensor Data Through REST API

Get the real-time sensor data from a REST server.

var rest = new iotjs.Rest();
// Add the air quality sensor instance into a REST router.
rest.route('/aq/data', aq);
rest.listen(8080, function() {
  console.log('Air quality station is listening on port 8080!');
});

http://device.domain.name:8080/aq/data is the REST API to get the latest sensor data.
