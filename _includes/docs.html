Getting Started

Install IoT.JS


$ npm install iotjs

PM2.5 Air Quality Station Send Data to Data Cloud

Code for a PM2.5 air quality station, send data to a data cloud service through REST API and hava capability of accessing the sensor data throught REST API automatically generated by IoT.JS.


var request = require('request');
var iotjs = require('iotjs');
// Define a PM2.5 sensor instance.
var pm25 = new iotjs.Sensors.PM25({ type: 'pms3003', rate: 60000 });

// Send data to a data cloud.
pm25.on('data', funciton(data) {
  request.post('REST_API_URI',
    { form: { pm25: data.pm25 } },
    function(err, response, body) {
      !err ? console.log(body) : console.error(err);
    });
});

// Run REST server.
iotjs.rest({ route: '/pm25/data', model: pm25}).listen(8080, function () {
  console.log('PM2.5 app is listening on port 8080!');
});

Sensor Data Through REST API

The REST API URI of getting sensor data is http://device.domain.name:8080/pm25/data.
