Getting Started

Blink an LED without any JavaScript code.

1. Install IoT.js

$ npm install iotjs

2. Setup board.json

Add a new file board.json and add the below JSON object.

{
  "modules": {
    "led" : { "module": "Led", "pins": 44 }
  },
  "extensions": {
    "thingjs-blink": { "modules": "led" }
  }
}

3. Run

Run the below command in terminal.

~/path/to/iotjs/iotjs
